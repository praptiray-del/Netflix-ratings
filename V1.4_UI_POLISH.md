## 📐 New Layout - Search Results

### Before (V1.3):
```
Grid of large cards:
┌─────────┐ ┌─────────┐ ┌─────────┐
│ [HUGE  │ │ [HUGE  │ │ [HUGE  │
│  IMAGE]│ │  IMAGE]│ │  IMAGE]│
│  Title │ │  Title │ │  Title │
│  Info  │ │  Info  │ │  Info  │
└─────────┘ └─────────┘ └─────────┘
```

### After (V1.4):
```
List of compact rows:
┌────────────────────────────────────────┐
│ [img] Title                → Details  │
│       ⭐ 8.5  📅 2015  🎭 Movie        │
├────────────────────────────────────────┤
│ [img] Title                → Details  │
│       ⭐ 7.2  📅 2020  🎭 Series       │
├────────────────────────────────────────┤
│ [img] Title                → Details  │
│       ⭐ 9.1  📅 2018  🎭 Movie        │
└────────────────────────────────────────┘
```

Much more efficient! See more results without scrolling!
